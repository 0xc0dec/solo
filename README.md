## About
A playground for my graphics/gamedev experiments.

## Features
* Unity-inspired component system
* OBJ meshes and PNG textures loading
* Asynchronous asset loading
* Render to texture
* Text rendering
* Physics
* OpenGL rendering, Vulkan is coming
* Fully scriptable (Lua)
