## About
A playground for my graphics/gamedev experiments.

## Features
* Unity-inspired component system
* Loading of OBJ meshes and several popular texture formats
* Render to texture
* TTF fonts
* Physics
* Vulkan/OpenGL rendering
* Fully scriptable (Lua)
